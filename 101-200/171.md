### 171. Excel表列序号

难度：Easy

相关话题：`数学`

给定一个Excel表格中的列名称，返回其相应的列序号。



例如，



```
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
```


**示例 1:** 



```
输入: "A"
输出: 1
```


**示例2:** 



```
输入:"AB"
输出: 28
```


**示例3:** 



```
输入:"ZY"
输出: 701
```

```
/**
 * @param {string} s
 * @return {number}
 */
var titleToNumber = function(s) {
  if(s==='')return 0
  return (s.charCodeAt(0)-65+1)*Math.pow(26,s.length-1) + titleToNumber(s.slice(1))
};
```

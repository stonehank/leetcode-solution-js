### 172. 阶乘后的零

难度：Easy

相关话题：`数学`

给定一个整数 *n* ，返回 *n* ! 结果尾数中零的数量。



**示例 1:** 



```
输入: 3
输出: 0
解释:3! = 6, 尾数中没有零。
```


**示例2:** 



```
输入: 5
输出: 1
解释:5! = 120, 尾数中有 1 个零.
```


**说明:** 你算法的时间复杂度应为*O* (log*n* )** 。


```
/**
 * @param {number} n
 * @return {number}
 */
var trailingZeroes = function(n) {
  let x=5,res=0
  while(x<=n){
    res+=Math.floor(n/x)
    x*=5
  }
  return res
};
```

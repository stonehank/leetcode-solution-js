### 611. 有效三角形的个数

难度：Middle

相关话题：`数组`

给定一个包含非负整数的数组，你的任务是统计其中可以组成三角形三条边的三元组个数。



**示例 1:** 



```

输入: [2,2,3,4]
输出: 3
解释:
有效的组合是: 
2,3,4 (使用第一个 2)
2,3,4 (使用第二个 2)
2,2,3
```


**注意:** 




1. 数组长度不超过1000。

2. 数组里整数的范围为 [0, 1000]。




```
/**
 * @param {number[]} nums
 * @return {number}
 */
var triangleNumber = function(nums) {
    let sum=0
    nums.sort((a,b)=>a-b)
    for(let i=nums.length-1;i>=2;i--){
        let l=0,r=i-1
        while(l<r){
            if(nums[l]+nums[r]>nums[i]){
                sum+=r-l
                r--
            }else
                l++
        }
    }
    return sum
};
```

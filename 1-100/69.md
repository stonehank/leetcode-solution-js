### 69. x 的平方根

难度：Easy

相关话题：`数学`、`二分查找`

实现 `int sqrt(int x)` 函数。



计算并返回 *x* 的平方根，其中 *x * 是非负整数。



由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。



 **示例 1:** 





```
输入:

 4
输出:

 2

```

 **示例 2:** 





```
输入:

 8
输出:

 2
说明:

 8 的平方根是 2.82842..., 
    由于返回类型是整数，小数部分将被舍去。

```


```
/**
 * @param {number} x
 * @return {number}
 */
var mySqrt = function(x) {
  if(x===1)return 1
  let l=0,h=Math.floor(x/2)
  while(l&lt;h){
    let mid=Math.floor((l+h)/2)+1
    if(mid*mid&gt;x)h=mid-1
    else l=mid
  }
  return l
};



```

### 961. 重复 N 次的元素

难度：Easy

相关话题：`哈希表`

在大小为  `2N` 的数组  `A` 中有  `N+1`  个不同的元素，其中有一个元素重复了  `N`  次。



返回重复了  `N` 次的那个元素。













 **示例 1：** 





```
输入：

[1,2,3,3]
输出：

3

```

 **示例 2：** 





```
输入：

[2,1,2,5,3,2]
输出：

2

```

 **示例3：** 





```
输入：

[5,1,5,2,5,3,5,4]
输出：

5

```





 **提示：** 





1.  `4 &lt;= A.length &lt;= 10000` 

2.  `0 &lt;= A[i] &lt; 10000` 

3.  `A.length` 为偶数






```
/**
 * @param {number[]} A
 * @return {number}
 */
var repeatedNTimes = function(A) {
    let hash={}
    for(let i=0;i&lt;A.length;i++){
      let cur=A[i]
      if(hash[cur]==null)hash[cur]=true
      else return cur
    }
};



```
